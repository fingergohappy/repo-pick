# Skill Creator

Use when users want to create a new skill...

## Workflow
1. Clarify purpose
2. Draft the skill
3. Verify behavior
h/l move focus · j/k move item · H left pane · L right pane · ? shortcuts · q quit target: .codex/skills · force: off

需要确认的一个键位语义

A

O = 载入/搜索选中的仓库

在左栏选中 registry 后，按 O clone/search 该仓库，并刷新中间 skill 列表。

B

O = 用系统浏览器打开仓库 URL

TUI 只负责跳到外部仓库页面，搜索仍用 enter 或其他键触发。
