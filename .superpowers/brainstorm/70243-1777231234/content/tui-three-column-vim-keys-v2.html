# Skill Creator

Use when users want to create a new skill...

## Workflow
1. Clarify purpose
2. Draft the skill
3. Verify behavior
h/l move focus · j/k move item · H left pane · L right pane · ? shortcuts · q quit target: .codex/skills · force: off

下一处需要确认：自动载入策略

A

推荐：停留后自动载入

j/k 快速移动时不立刻 clone；光标停留短暂延迟后载入，避免连续触发多个仓库搜索。

B

立即自动载入

每次移动到 registry 都立刻搜索。反馈直接，但远程仓库多时可能比较吵，也容易浪费 clone。
