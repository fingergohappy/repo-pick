# Skill Creator

Use when users want to create a new skill...
tab switch pane · enter load/search · space select · / filter · q quit target: .codex/skills · force: off

安装动作放置方式

请选择第一版更适合的安装确认方式。

A

底部快捷键安装

按 i 打开安装确认弹窗，确认目标目录和是否覆盖。主界面保持浏览效率。

B

右侧预览底部固定按钮区

右栏底部长期显示 Install / Force / Target 信息。更显眼，但会挤占 preview 空间。

C

选择后进入安装步骤

从浏览界面切到安装确认页，适合复杂配置；第一版流程会稍重。
